# Sales Data Analysis Project (Python)

## Objective
Analyze retail sales data to uncover business insights using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Key Analysis
- Total Revenue & KPIs
- Best-selling products
- Monthly sales trends
- City-wise performance
- Category analysis

## Business Insights
- Laptop is the top revenue-generating product
- February shows peak sales
- Chicago leads in revenue
- Electronics outperform accessories

## Skills Demonstrated
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insight Generation
